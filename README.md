# Lab - VLAN Segmentation and VoIP Network

## Overview

This project was developed using Cisco Packet Tracer to simulate an enterprise network focused on VLAN segmentation and VoIP integration.

The network is divided into four departments, each assigned to its own VLAN, improving network organization, performance, and security. A dedicated Voice VLAN was also configured to support IP telephony across the enterprise.

---

## Objectives

- Configure multiple VLANs
- Implement a dedicated Voice VLAN
- Connect IP Phones to each department
- Configure inter-VLAN routing
- Improve network segmentation
